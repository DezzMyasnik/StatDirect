Для добавления аккакунтов нужно нажать на кнопку плюса.
В появившемся окне заполнить поля аккаунта и сайта который рекламирует этот аккаунт. 
ВАЖНО!!! эти поля сугубо информационные и нужны для последующей корректной работы программы. 
Для авторизации аккаунта нужно нажать на кнопу №1 Авторизация. 
При первом запуске программы загрузиться страница яндекса, накоторой надо ввести логин и пароль от ребуемого аккаунта,
встроенный браузер будет перенаправлен на странице разрешения доступа к приложению, нужно нажать кнопку "Разрешить", 
дальше браузер будет перенаправлен на финальную страницу. На ней обычно ни чего не отображается кроме серенкого меню и кнопки "вход", 
эта стнаица в адресе содержит токен.  
нажать на кнопу №2. получить токен, в сером поле отобра.
нажать кнопку №3 Добавить в базу. в поле ниже отобразиться логин аккаунта, что означает что все данные заненсены в базу данных.

Для добавления следующего аккаунта нужно:
во встоенном браузере выйти из текущего аккаунта, для этого найти ссылку "Яндекс", обычно в правом нижнем углу страницы->перебросит на главную страницу яндекса-> выйти из аккаунта-> войти в новый аккаунт->заполнить поля логина и сайта-> нажать кнопку 1-> нажать кнопку 2-> нажать кнопку нопу 3.

после добавления всех аккаунтов закрыть окно
Сформирутеся таблица с пустыми полями. нажать кнопуку обновить, в результате будут выведены сводные данные о балансе и расходах за последние 7 дней.
получить данные о разходах за больший период почемуто не дает апи яндекса.
